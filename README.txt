SimplexMobile
Aplicativo para realização do calculo de Simplex.
Desenvolvido como projeto para a matéria de Programação Linear do curso de Analise e Desenvolvimento de Sistemas.
É necessário a instalação do apk SimplexMobile em um celular Android para a execução do aplicativo.
